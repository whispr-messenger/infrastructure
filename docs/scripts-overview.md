# Scripts

## Vue d'ensemble

Les scripts d'automation se trouvent dans le dossier `scripts/`.

| Script | Rôle |
|--------|------|
| `scripts/vault/init-vault.sh` | Initialiser Vault |
| `scripts/vault/populate-secrets.sh` | Peupler les secrets |
| `scripts/platform-engineers/` | Setup accès équipe |
